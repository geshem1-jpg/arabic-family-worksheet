אל-עילה (אִלְעֵילָה) - המשפחה

1. אוצר מילים בסיסי (הקניה)

שימו לב לכינוי השייכות "שלי" (סיומת ִי):

ערבית (תעתיק) עברית
אַבּוּי (Abuy)אבא שלי
אֻמִּי (Ummi)אמא שלי
אַח'וּי (Akhuy)אח שלי
אֻחְ'תִי (Ukhti)אחותי
סִידִי / גֶ'דִּיסבא שלי
סִתִּי / גֶ'דַּתִיסבתא שלי

2. תרגול כינויי שייכות

השלימו את המילה "אסם" (שם) עם הסיומת המתאימה:

השם שלו: אִסְמוֹ

השם שלה: (תשובה: אִסֵמְהַא)

השם שלך (זכר): (תשובה: אִסְמַכּ)

3. מה הם עובדים? (מַדַא בִּיִשְׁתְעִ'לּוּ?)

תרגמו את המקצועות הבאים (היעזרו בעמ' 19):

  • מורה (זכר): (מֻעַלִּם)
  • רופאה: (דֻכְּתוֹרָה)
  • שוטר: (שֻׁרְטִי)

4. הבנת הנקרא: עֵילַת ראני (משפחת ראני)

אִסְמִי רַאני, וַאַנַא מִן עֵילֶת מַנְסוּר. עֻמְרִי תְנַעְשַׁר סַנֶה.
אַנַא סַאכֵּן מַע עֵילְתִי פִי מַדִינַת חֵיפַא.
אִסְם אַבּוּי יוּסֵף, וְהֻוַ רַגֻ'ל אַעְמַאל. אִסְם אֻמִּי סַלְמַא, וְהִיַ מֻמַרִּדַ'ה פִי מֻסְתַשְׁפַא.
